Arduino Nokia Snake Game

A fun remake of the classic Nokia Snake using Arduino, OLED, and joystick.

Download Code

Features

  • Classic Nokia style gameplay
  • Joystick control and pause functionality
  • OLED display support with sound feedback
  • High score saved in EEPROM
  • Adjustable speed and simple on-screen menu
